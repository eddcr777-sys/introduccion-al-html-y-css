¿Qué es CSS?

CSS es un lenguaje que proporciona mecanismos para aplicar estilos a los elementos HTML en una página web. Facilita la creación de un diseño visual atractivo y uniforme a través de propiedades que permiten definir cómo se mostrarán los diversos elementos en diferentes dispositivos y resoluciones de pantalla.

Su Historia

Historia de CSS La evolución de CSS ha sido crucial para el crecimiento y la funcionalidad de la web. Aquí se resumen las etapas clave en la historia de CSS: 1. CSS1 (1996) La primera versión, CSS1, fue publicada por el W3C en diciembre de 1996. Estableció una serie de propiedades básicas para el control del diseño visual, como colores, fuentes y márgenes. 2. CSS2 (1998) CSS2 fue lanzado en mayo de 1998 e introdujo nuevas características como el soporte para posicionamiento, media queries (consultas de medios) y estilos para documentos impresos. 3. CSS2.1 (2004) CSS2.1 fue una revisión de CSS2 para corregir problemas y clarificar especificaciones. Se convirtió en un estándar más robusto para los desarrolladores. 4. CSS3 (2009-presente) CSS3 se desarrolló de manera modular, lo que permitió la introducción de nuevas características y propiedades de forma más rápida. Incluye módulos para selecciones, bordes, sombras, gradientes, animaciones y flexbox. Con CSS3, se mejoró la capacidad de crear diseños sofisticados y responsivos.

Estructura

El CSS se puede incluir en un documento HTML de tres maneras:

Inline

Estilos aplicados directamente en los elementos HTML mediante el atributo "style":

<p style="color: red;">Este es un párrafo rojo.</p>

Interno

Estilos definidos dentro de una etiqueta <style> en el <head>:

<head>
    <style>
        p {
            color: blue;
        }
    </style>
</head>

Externo

Estilos vinculados a un archivo CSS externo mediante la etiqueta <link> en el <head>:


    <head>
        <link rel="stylesheet" href="estilos.css">
    </head>

Selectores

Los selectores permiten aplicar estilos a elementos específicos en función de sus etiquetas, clases, identificadores o atributos.

Tipo De Selector Ejemplo Descripcion
Selector de etiqueta p { color: red; } Selecciona todos los parrafos
Selector de Clase .mi-clase { font-size: 16px; } Selecciona todos los elementos con la clase "mi-clase"
Selector de Identificador #mi-elemento { background-color: blue; } Selecciona el elemento con el identificador "#mi-elemento"
Selector de Atributo [type="text"] { border: 1px solid black; } Selecciona elementos con un atributo específico.
Selectores Compuestos h1, h2, h3 { margin: 0; } Selecciona todos los elementos h1, h2 y h3.
Selectores Descendientes ul li { list-style-type: none; } Aplica estilos a elementos hijos.

Las propiedades son características que definen el estilo de los elementos. Algunas de las propiedades más comunes incluyen:

Propiedad Descripción
color Define el color del texto.
background-color Define el color de fondo.
font-family Especifica la fuente de texto.
font-size Define el tamaño de la fuente.
text-align Alinea el texto.

padding Agrega espacio interno al elemento.
margin Agrega espacio externo al elemento.
width Define el ancho del elemento.
height Define la altura del elemento.
display Define cómo se mostrará el elemento.
position Define la posición del elemento.
top Define la posición vertical del elemento.
left Define la posición horizontal del elemento.

Box Model

El modelo de cajas es un concepto fundamental en CSS que describe cómo se estructuran y representan los elementos en una página web. Cada elemento HTML es representado como una caja rectangular, y este modelo define cómo se gestionan el margen, el borde, el relleno y el contenido.

Componentes del Modelo de Cajas:

Contenido:
Es el área donde se encuentra el texto, las imágenes o cualquier otro elemento. Las dimensiones de esta área se pueden controlar con las propiedades width y height.

Padding (Relleno):

El espacio entre el contenido y el borde de la caja. El padding se define con las propiedades padding-top, padding-right, padding-bottom y padding-left.

Border (Borde):

puede definir de manera individual para cada lado: border-top, border-right, border-bottom y border-left Impacta en el tamaño total de la caja.

Margin (Margen):

El espacio fuera del borde que separa el elemento de otros elementos. Al igual que el padding, se puede definir para cada lado de manera independiente: margin-top, margin-right, margin-bottom y margin-left.

Cálculo del Tamaño Total

Para calcular el tamaño total de un elemento en el modelo de cajas, se debe sumar el contenido, el relleno, el borde y el margen:

Tamañototal=width+padding(izq+der)+border(izq+der)+margin(izq+der)

Ejemplo:

.box {

    width: 200px;              /* Ancho del contenido */

    padding: 20px;            /* Espacio interno */

    border: 5px solid black;  /* Borde */

    margin: 10px;            /* Espacio externo */

}

En este caso, el tamaño total de la caja será: 200+20×2+5×2+10×2=200+40+10+20=280px

Flexbox

Flexbox (Flexible Box Layout) es un modelo de diseño en CSS que facilita la creación de diseños unidimensionales, permitiendo organizar los elementos dentro de un contenedor de manera flexible. Este modelo es especialmente útil para centrar contenido y distribuir espacio entre elementos de forma dinámica.

Propiedades Clave de Flexbox:

Contenedor Flex:

Para activar el flexbox, se debe asignar display: flex o display: inline-flex; al contenedor, lo que convierte todos los elementos hijos en elementos flexibles.

Ejemplo:
.contenedor {
    display: flex;
}

flex-direction:

Define la dirección de los elementos en el contenedor. Puede tomar valores como:
Valor Descripción
row (predeterminado): Los elementos se organizan en fila.
column: Los elementos se organizan en columna.
row-reverse: Los elementos se organizan en fila pero en orden inverso.
column-reverse: Los elementos se organizan en columna pero en orden inverso.
Ejemplo:
.contenedor {
    flex-direction: row; /* o column, row-reverse, column-reverse */
}

justify-content:

Alinea los elementos a lo largo del eje principal (horizontal o vertical dependiendo de flex-direction):

Valor Descripción
flex-direction: Alinea al inicio.
flex-end: Alinea al final.
center: Alinea al centro.
space-between: Espacio igual entre los elementos.
space-around: Espacio igual alrededor de los elementos.
Ejemplo:
.contenedor {
    justify-content: center; /* o flex-start, flex-end, space-between, space-around */
}

align-items:

La propiedad align-items alinea los elementos flexibles a lo largo del eje transversal (vertical si la dirección es row, horizontal si es column). Aquí están los valores más comunes:

valor descripcion
flex-start: Alinea al inicio.
flex-end: Alinea al final.
center: Alinea al centro.
stretch: Los elementos ocupan toda la altura del contenedor flex.
baseline: Los elementos se alinean según la línea base de su texto.
Ejemplo:
.contenedor {
    align-items: center; /* o flex-start, flex-end, baseline, stretch */
}

Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite organizar elementos en columnas y filas, proporcionando un control completo sobre el diseño de la página web. Es especialmente útil para crear layouts complejos y responsivos de manera eficiente.

Propiedad Descripción Ejemplo
grid-template-columns: Define las columnas del grid. .contenedor { display: grid; }
grid-template-rows: Define las filas del grid. .contenedor { grid-template-columns: 1fr 2fr; /* Dos columnas: 1 parte y 2 partes */ grid-template-rows: auto 100px; /* Dos filas: primera automática y segunda de 100px */ }
grid-template-areas: Define las áreas del grid. .contenedor { gap: 10px; /* Espacio entre elementos */ }
grid-gap: Define el espacio entre los elementos. .elemento { grid-area: cabecera; /* Coloca este elemento en la área 'cabecera' */ }
